# Bug Report — Login Error

**Bug ID:** BUG_LOGIN_001  
**Severity:** High  
**Priority:** High  

## Description
User is unable to login even when using valid credentials.

## Steps to Reproduce
1. Navigate to login page
2. Enter a valid email address
3. Enter the correct password
4. Click the login button

## Expected Result
User should be logged into the system successfully.

## Actual Result
System displays an error message and prevents login.

## Environment
Browser: Google Chrome  
OS: Windows 10  

## Notes
This issue prevents users from accessing their accounts.
